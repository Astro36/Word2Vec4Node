# Word2Vec4Node

> Google Word2Vec Interface for Node.js with N-API

## Notice

Only **Unix-like** OS can use `Word2Vec.train()` method. [pthread](https://en.wikipedia.org/wiki/POSIX_Threads) only working on POSIX, Windows can not use that method.
